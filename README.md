## **Introduce**
- **PaP(Pet and Pet)** 프로젝트는 반려인이 반려동물들의 산책 메이트를 찾기 위한 커뮤니티 서비스 플랫폼입니다.
- 반려인들이 서로의 반려동물들의 사진들을 공유하여 좋아요와 즐겨찾기를 활용하여 다른 반려인들과의 소통을 도와줍니다
